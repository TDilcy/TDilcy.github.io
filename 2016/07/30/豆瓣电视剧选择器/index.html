豆瓣电视剧选择器

发表于 2016-07-30   |   分类于 python ， crawler   |  

获取豆瓣电视剧信息

简介

输入电视剧类型（美剧，英剧，韩剧，日剧，港剧，国产剧，日本动画）、排名方法（推荐，时间，评分）、欲输出的数目，即可得到所需信息。

code

学习内容

  • 使用selenium自动化浏览器，模拟其操作实现动态网页（及包含javascript）的抓取，包括元素选取，点击按钮，设置等待。
  • 使用selenium选择页面元素是采用了find_element_by_xpath方法，初步学习了下path语法。
  • 使用了headless浏览器PhantomJS代替Chrome。

注意事项及其他

  • 编码问题，勿对str进行encode，勿对unicode进行decode。
  • 调用Chrome需使用ChromeDriver，在调用Chrome及PhantomJS时需将其添加至系统环境变量，若不添加，则在调用时需写明路径。
  • 在抓取某些在部分电影存在，而在其他电影未必存在的信息，需采用异常处理机制，若检查出不存在，则需要用其他信息替换。

问题及不足

  • excel读取csv文件时易出现乱码，考虑改还信息存储方式（如使用MySQL）
  • 输入检查未编写完成，而且目前只能考虑检查输入类型是否正确（是否输入数字），应考虑到输入范围。
  • 获取集数和季数
